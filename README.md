# pytest-Quick-Start-Guide
pytest Quick Start Guide, published by Packt

## Run Examples

* Create a virtual envirnment:

  ```
  python3 -m venv .env
  source .env/bin/activate
  ```

* Install requirements:
  
  ```
  pip install -r requirements.txt
  ```

* Install `pre-commit`  :

  ```
  pre-commit install
  ```
